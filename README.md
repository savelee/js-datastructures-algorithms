## Directory Structure

* src/ts/datastructures - Data Structures in TypeScript
* src/examples/ - Implementation examples. 
* tests/ - Unit Tests to verify written code

## Setup

Run `npm install`

## Test

Run `npm run test`

## Start

Run `npm start`
Visit http://localhost:3000
